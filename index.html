Mouse events

"O" - Set mouse orbit on mouse position
Right click - Select element
Shift + Right click - Multi selection
Shift + Mouse dragging - Box selection

Visibility events

"H" - Hide selected elements
"I" - Isolate selected elements
"C" - Clear hidding/isolating

Clipping events

"P" - Create clipping plane on mouse position
Alt + "P" - Remove clipping planes
